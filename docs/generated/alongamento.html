🧘 Alongamento de Quadril (Borboleta)

Abre quadril para manobras. Posição sentada com plantas dos pés juntas, joelhos para fora. Segure 30 segundos.

📋 Passo a passo:

  1. Sente no chão, costas retas
  2. Junte plantas dos pés, deixando joelhos caírem para os lados
  3. Segure pés com as mãos
  4. Incline corpo para frente mantendo coluna reta
  5. Segure 30 segundos, respire fundo

✨ Benefícios:

Maior mobilidade para bottom turnPrevine lesões no quadrilMelhora abertura para manobras
