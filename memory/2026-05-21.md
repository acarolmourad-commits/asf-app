## 🧠 Aprendizados

- **WSL path confusion**: Quando o usuário acessa arquivos pelo WSL com `/c/Users/`, `~` expande para `/c/Users/Carolina/` — não é o mesmo que `/root/` no servidor OpenClaw.
- **Multi-environment apps**: Quando há dois ambientes de trabalho, sempre confirmar o caminho ABSOLUTO antes de modificar arquivos.
- **Script tag balanceamento no HTML inline**: Arquivos SPA monofile com `<script src="...">` requerem análise cuidadosa — o `src` causa o browser ignorar todo o conteúdo inline até `</script>`.
- **sed / NNr strategy para arrays**: Inserção de múltiplas linhas APÓS o último item do array com `sed '/last-item/ r tempfile'` — mais confiável que `a` ou `i` para edições de arrays JavaScript.
- **Skipping interrupções automáticas**: Mensagens `⚡ Interrupting current task (iteration 1/90)` sem mensagem humana são **false positive do sistema** — responder `NO_REPLY`, não entrar em loop.
- **hermes_tools é Windows-only**: Módulo `from hermes_tools import terminal` só existe no ambiente Hermes/Windows — usar ferramentas nativas (`curl`, `grep`, `sed`, heredoc Python) no servidor OpenClaw.
